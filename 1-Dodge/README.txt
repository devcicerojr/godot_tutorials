# Dodge

This game was coded following the instructions on Godot3_dodge tutorial: "Dodge the Creeps!"  Godot Engine 3.0 "getting started" game demonstrating the basics.


The tutorial is a companion to the Godot Engine 3.0 Getting Started Guide (link not yet available).
It is a game intended to demonstrate the following fundamental concepts of Godot development:

*  Node and scene structure
*  Instancing scenes
*  Input
*  Signals
*  UI
